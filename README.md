#Core - Last Stand
==========

Last Stand is a WIP 2D shooter.

###Controls:
* Space - Fire
* W - Move Up
* S - Move Down
* A - Move Left
* D - Move Right
* LCTRL -  Shield

###Dependencies
1. [LWJGL 2.8.5](http://www.lwjgl.org)
2. [jackson 2.2.0](http://jackson.codehaus.org)
3. [Core 0.5.1](http://www.github.com/jgefroh/core)

Have fun!
